Functions or Methods.

In Python, a function is a block of code that performs a specific task and can be called by other parts of a program. Functions in Python are a way to modularize code, making it easier to read, write, and maintain. Functions can have inputs (arguments) and outputs (return values), or they can perform actions without returning anything.

Functions are defined using the def keyword, followed by the name of the function, parentheses that may include arguments, and a colon. Here's a breakdown of the different parts of a function:
def keyword:: This indicates that a function is being defined.
function_name: This is the name of the function. It should be a descriptive name that indicates what the function does.
parameters: These are optional input values that the function can take. Parameters are enclosed in parentheses, and multiple parameters are separated by commas.
docstring: This is an optional string that describes what the function does. It is typically included as a comment immediately after the function definition.
code block:: This is the actual code that the function executes when it is called.
return: This statement is used to return a value or values from the function. It is optional, and if it is not included, the function will return None by default. A function can return multiple values as a tuple.
