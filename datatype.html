Data Types.

In Python, a data type is a classification of the type of value a variable can hold. Python has several built-in data types that can be used in a program. These include:

1. Numeric Types:
     a. int - integers (positive, negative or zero)
          example: 10, -8.
     b. float - floating-point numbers with decimals
          example: 10.10, -8.26.
     c. complex - numbers with a real and imaginary part
          example: 4+3i, 3-i.

2. Boolean Type:
     a. bool - represents a boolean value, True or False

3. Sequence Types:
     a. str - a string of characters, such as "hello world"
          example: "Computer Science"
     b. list - an ordered collection of values, such as [1, 2, 3]
          example: List of products availabe ["Laptop", "Mobile", "Earbuds", "Smart Watch"]
     c. tuple - an ordered, immutable collection of values, such as (1, 2, 3)
          example: Tuple representing full name ("Dusty","Rhodes")

4. Set Type:
     a. set - Set items are unordered, unchangeable, and do not allow duplicate values., such as {1, 2, 3}
          example: my_set = {1,2,3,4,5}.

5. Dictionary Type:
     A dictionary is a built-in data type that represents a collection of key-value pairs.
          example: my_dict = {"name": "Dan", "age": 33, "profession": "singer"}
