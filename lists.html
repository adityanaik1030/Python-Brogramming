List.

In Python, a list is a collection of ordered, mutable, and heterogeneous items enclosed in square brackets []. It can contain elements of any data type such as integers, floats, strings, and even other lists. Lists are one of the most commonly used data structures in Python as they provide a convenient way to store and manipulate data.

To create a list, we can simply enclose a comma-separated sequence of items in square brackets:



Once we have created a list, we can perform various operations on it such as adding or removing elements, accessing elements, slicing, iterating, and more.


Adding Elements.

We can add elements to a list using the append() method, which adds an element to the end of the list. We can also use the extend() method to add multiple elements to a list.




Removing Elements.

To remove elements from a list, we can use the remove() method, which removes the first occurrence of the specified element. We can also use the pop() method to remove an element at a specific index, or simply use the del statement to remove a slice of elements.




Accessing Elements.

We can access the elements of a list by using their index values. The index of the first element is 0, the second is 1, and so on. We can also use negative indexing to access elements from the end of the list.
